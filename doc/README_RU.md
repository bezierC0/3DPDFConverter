# 3DPDFConverter

[English](../README.md) | [中文](README_CN.md) | [日本語](README_JP.md) | [한국어](README_KO.md) | [русский](README_RU.md)

Это модульная платформа для генерации 3D PDF, основанная на C++, OpenCASCADE и libHaru.

## Особенности
*   **Парсинг STEP**: Чтение файлов в формате STEP/STP.
*   **Построение сеток**: Генерация треугольных сеток с использованием алгоритма `BRepMesh_IncrementalMesh` из OpenCASCADE.
*   **Конвертация PRC**: Преобразование данных сетки в формат PRC для обмена 3D-данными.
*   **Экспорт PDF**: Встраивание данных PRC с использованием libHaru и вывод стандартных 3D PDF.
*   **Высокая степень разделения**: Основные функции разделены на несколько модулей DLL (StepReader, MeshGenerator, PrcExporter, PdfExporter), предоставляя безопасные интерфейсы в стиле C для легкой интеграции с различными языками и наборами инструментов.

## Структура модулей
- `CoreTypes`: Определяет чистые структуры C и дескрипторы интерфейсов для связи между DLL.
- `Logger`: Потокобезопасный компонент для ведения логов.
- `StepReader`: Загружает и анализирует модели STEP.
- `MeshGenerator`: Триангулирует BRep из STEP в сетку.
- `PrcExporter`: Генерирует бинарный поток PRC.
- `PdfExporter`: Генерирует итоговый PDF-файл и встраивает 3D-вид.
- `ExportFacade`: Предоставляет единый интерфейс-обертку для экспорта в один клик.
- `CLI`: Точка входа для запуска из командной строки (`3DPDFConverterCLI.exe`).

## Инструкции по сборке
Этот проект использует CMake и vcpkg для управления зависимостями:
```bash
# Генерация проекта (включая зависимости, такие как GTest, libharu)
cmake -S . -B build -G Ninja -DCMAKE_TOOLCHAIN_FILE=[Ваш путь к vcpkg]/scripts/buildsystems/vcpkg.cmake

# Сборка
cmake --build build
```
